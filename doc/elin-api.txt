*elin-api.txt*    API for |elin|                                    *elin-api*

Each api method has kebab and camel case. The latter makes it easier in lua.

                                                            *elin.compileAll*
(elin.compile-all {opts})                                   *elin.compile-all*
  One-shot compile all `.fnl` files to |elin-rtp|.

  Parameters: ~
    • {opts}  (`table<string, any>`) Path

  Return: ~
      (`[bool,string?,string,string?][]`) list with compilation statuses.
      Each element is a tuple with the following fields:
        • {ok} (`boolean`) `true` if compilation succeeded, else `false`
        • {err} (`string?`) error message not {ok}
        • {in} (`string`) in `.fnl` file elin tried to compile
        • {out} (`string?`) out `.lua` file if {ok} = `true`, else `nil`
